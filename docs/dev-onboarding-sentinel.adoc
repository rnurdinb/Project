== Local Machine Setup

. _Install_ NPM dan nodejs
. _Clone_ dari git https://github.com/sepulsa/kraken-admin.git[]
. Pindah ke _app directory_ dan jalankan "npm install"
. _Run_ app menggunakan command "npm run dev"
